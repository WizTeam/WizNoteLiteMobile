# WizNote Lite for Mobile (iOS / Android, phone or pad)

WizNote Lite React Native Project

## install

### instal Xcode 11 or later

```bash
npm i
cd ios
pod install
cd ..
cd web_editor
npm i
npm run build
cd ..
```

run on iPhone

```bash
npm run ios
```

run on iPad

```bash
npx react-native run-ios --simulator="iPad (7th generation)"
```
